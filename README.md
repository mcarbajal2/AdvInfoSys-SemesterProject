# AdvInfoSys-SemesterProject
Solo term research project (Spring 2026) at Clayton State University which aims to understand trade-offs between power, efficiency, and cost in a cloud computing environment. 

The CloudSim Plus framework was utilized to simulate data centers of differing power levels, with the purpose of comparing them on energy and cost evaluation metrics.

Results were later exported into Power BI to create data visualizations.

## Project Timeline
- **Week 1** (March 15–21): <br>Install CloudSim Plus and successfully run a test program.
- **Weeks 2–3** (March 22–April 4): <br>Research appropriate simulation environment parameters.
- **Weeks 4–5** (April 5–April 18): <br>Run simulations with changing cloud infrastructure parameters and measure the results.
- **Week 6** (April 19–25): <br>Evaluate each simulation on how much power was saved vs. how much speed was lost.
- **Week 7** (April 26–May 2): <br>Final review and project submission.
